Mô tả:
    I. Cách thức cài đặt/biên dịch/chạy chương trình
        1. Mở project bằng Eclipse và Run.
        2. Mở trực tiếp bằng file jar: double click vào file jar để chạy chương trình.
        3. Mở bằng terminal/command prompt: cd đến root của project. Chạy: java -jar FlowFree.jar.
    II. Gói phần mêm, thư viện kèm theo: AbsoluteLayout và Java Swing.
